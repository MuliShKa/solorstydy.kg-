# SolarStudy — Landing Page

> Educational agency website for students applying to universities in South Korea, USA and Europe.

**Live:** [solarstudy.kr](https://www.instagram.com/solarstudy.kr/)

---

## Tech Stack

| Layer | Technology |
|---|---|
| Markup | HTML5 |
| Styling | CSS3 (custom, no framework) |
| Scripts | Vanilla JS + jQuery 3.6 |
| Slider | Swiper.js + Owl Carousel 2 |
| Icons | Font Awesome 6 |
| Fonts | Google Fonts — Montserrat, Playfair Display, Raleway |

## Features

- Responsive hamburger navigation (pure CSS)
- Image gallery with Swiper.js slider
- Floating social buttons (WhatsApp, Telegram, Instagram, YouTube)
- Interactive profile card with tab sections (About / Countries / Contacts)
- Smooth scroll & animations

---

## Getting Started

```bash
git clone <repo-url>
cd solarstudy
# open index.html in browser — no build step needed
```

No dependencies to install — all libraries are loaded via CDN.
